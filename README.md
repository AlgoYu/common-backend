# 自用后台管理界面
自用后台管理模板。
## 项目技术
1. **Vue**
2. **Vue-router**
3. **Vuex**
4. **ElementUI**
5. **Vue-Axios**
6. **Vue-Echarts**
7. **Vue-Amap**
8. **XSS**

## 使用方法
> 提醒：确保已经安装Node.js以及包管理工具NPM

1. 克隆仓库

```bash
git clone https://github.com/QQ794763733/Base-RBAC-BackEnd.git
```
2. 运行代码

```bash
npm run serve
```
